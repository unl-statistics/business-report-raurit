Appendix A — Draft: Data Documentation

For the exploration of the scenario “London Bridge is Falling Down”, the data set consulted will be pulled from the United States Department of Transit’s National Bridge Inventory. Not every variable in the data set will be included or transformed into a different variable. The data will pertain only to Douglas County in Nebraska, filtered with the known state and county codes. “National Bridge Inventory” (n.d.)
